# 요구사항 정리

## 출처
- Slack #아이디어 채널 (2026-04-01)
- 아이디어 구체화 스레드

## 사용자 스토리

### 1. 작년 일정 가져오기
- 사용자가 엑셀/CSV 파일을 업로드하면 작년 업무 일정을 자동 파싱
- 별도 입력 없이 기존 데이터 재활용
- 생산문서등록대장 CSV 형식 지원 (나이스 등에서 내보내기)

### 2. 비교 뷰
- 작년 내 일정 vs 올해 학교 공식 일정을 나란히 비교
- 반복 패턴 자동 감지 및 하이라이트
  - 예: "매년 3월 학급편성", "매년 4월 학교운영위원회" 등
- 날짜 차이 자동 계산 (올해 해당 시기 추천)

### 3. 검토 후 확정
- 가져온 일정을 바로 등록하지 않음
- 검토 단계에서 필요한 것만 선택
- 날짜 조정 후 원클릭으로 올해 일정에 확정

### 4. 자체 일정 관리
- 앱 자체적으로 일정 관리 기능 내장
- Google Calendar 연동은 선택사항 (추후)
- 외부 연동 없이도 독립적으로 사용 가능

## 데이터 분석 (샘플 기반)

### CSV 구조
| 컬럼 | 설명 | 활용도 |
|------|------|--------|
| 등록일자 | YYYY-MM-DD 형식 | 핵심 — 시기 패턴 분석 |
| 제목 | 문서 제목 | 핵심 — 업무 내용 파악 |
| 과제명 | 업무 대분류 | 카테고리 분류 |
| 과제카드명 | 업무 소분류 | 세부 카테고리 |
| 결재유형 | 생산/접수 | 내 업무 vs 수신 업무 구분 |
| 보존기한 | 5년/10년/30년 등 | 중요도 판단 참고 |

### 월별 분포 (2025년 샘플)
- 1월: 1건 (방학 중 최소)
- 2~3월: 신학기 준비 집중 (학급편성, 교육과정, 학교운영계획)
- 4~6월: 수업 운영, 행사, 학교운영위원회
- 7월: 여름방학 계획, 교육과정
- 8~10월: 2학기 운영, 인사, 예산
- 11~12월: 종업, 졸업, 차년도 준비

### 반복 업무 패턴 (매년 유사)
- 학교운영위원회 안건 (연 6~7회)
- 학년별 방학 계획
- 교육과정 계획 수립
- 학생 학적 관리 (반편성, 진급)
- 각종 품의 (호봉, 학교환경 등)
