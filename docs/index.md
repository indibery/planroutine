# 공직플랜 (PlanRoutine)

초등 교사를 위한 업무 일정 관리 앱입니다. 에듀파인 생산문서등록대장 CSV를
가져와 올해 일정을 빠르게 세팅하고, iOS 캘린더·Google 캘린더·로컬 알림까지
한 앱에서 관리합니다.

## 주요 기능

- **작년 일정 가져오기** — CSV 업로드로 작년 업무 사이클을 즉시 불러오기
- **검토 후 확정** — 슬라이드 한 번으로 일정 확정/삭제
- **자체 캘린더** — 앱 내 이벤트 CRUD + Google 캘린더 단방향 저장
- **로컬 알림** — 월초·1주 전·1일 전 08:00 자동 알림
- **공유시트 통합** — 카카오톡·메일·파일 앱에서 CSV를 공직플랜으로 바로 열기
